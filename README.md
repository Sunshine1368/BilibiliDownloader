# BiliBili Downloader

A Python application for downloading videos, danmaku (comments), and audio from Bilibili.

## Features

- Download Bilibili videos in various quality options
- Extract and save danmaku (bullet comments)
- Extract and download audio tracks
- Batch download support
- Progress tracking during downloads

## Requirements

- Python 3.7 or higher
- Required Python packages (see requirements.txt)

## Installation

1. Clone this repository:
```bash
git clone https://github.com/Sunshine1368/BiliBiliDownloader.git
cd BiliBiliDownloader
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the main program:
```bash
python BilibiliDownloader.py
```

Follow the prompts to:
- Enter the Bilibili video/playlist URL
- Select download options (video quality, include danmaku, extract audio)
- Choose output directory

## Configuration

Modify the settings in `BilibiliDownloader.py` to customize:
- Default download quality
- Output directory paths
- Danmaku format preferences
